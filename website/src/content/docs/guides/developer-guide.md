---
title: Developer Guide
---

1. Install all dependencies:

```zsh
pnpm install
```

If you are curious about the monorepository setup, you can read more about it here:

https://github.com/0x80/typescript-monorepo


2. Use the turbo commands to automatically build the upstream dependencies for your target subpackage.

3. Create a changeset

```sh
pnpm changeset
```
